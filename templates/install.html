Installation

A secret key is used as a factor to generate passwords. With different secret keys, erpass server is supposed to generate different passwords even thought master keys and options are the same. It should be generated when erpass is starting or be imported from another erpass server if you want to generate the same password.

The generator makes use of timestamps and SHA-256 to produce a new secret key which means a secret should be a 64 bytes string looks like 4355a4...0f3c9ee954a27460dd865.
